## Negation Instructions

### NOT - Logical Not
\tformat{0x17}

#### Desciption:
The `NOT` instruction performs a logical not operation on the source register and stores the result in the destination register.
Practically, this means that `rd` will be `1` if `rs` is `0` and `0` if `rs` is non-zero.

#### Action:
~~~
if rs = 0, rd := 1, else rd := 0
~~~

#### Assembler Syntax:
~~~
NOT {rd}, {rs}
~~~

#### Example:
~~~
; r0 = not r1
MODU r0, r1
~~~

### NEG - Integer Negate
\tformat{0x18}

#### Desciption:
The `NEG` instruction negates the source register and stores the result in the destination register, essentially multiplying `rs` by `-1`.

#### Action:
~~~
rd := -1 * rs
~~~

#### Assembler Syntax:
~~~
NEG {rd}, {rs}
~~~

#### Example:
~~~
; r0 = -1 * r1
NEG r0, r1
~~~

### INV - Bitwise Ivert
\tformat{0x19}

#### Desciption:
The `INV` instruction inverts all the bits in the source register and stores the result in the destination register.

#### Action:
~~~
rd := rs $\oplus$ 0xffffffff
~~~

#### Assembler Syntax:
~~~
INV {rd}, {rs}
~~~

#### Example:
~~~
; r0 = invert all bits of r1
INV r0, r1
~~~
