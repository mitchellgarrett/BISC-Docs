## Load Instructions

### LLI - Load Lower Immediate
\iformat{0x05}

#### Desciption:
The `LLI` instruction loads the given 16-bit immediate into the lower 16 bits of the destination register `rd`, zeroing out the top 16 bits.
Combined with the `LUI` instruction, you can load an entire 32-bit immediate into a register.

#### Action:
~~~
rd[0..15] := imm16
~~~

#### Assembler Syntax:
~~~
LLI {rd}, {imm16}
~~~

#### Example:
~~~
; load a decimal immediate
LLI r0, 512

; load a hexadecimal immediate
LLI r1, 0x42

; load a binary immediate
LLI r2, 0b11001100

; load a character
LLI r3, 'X'

; load a 16-bit immediate
LLI r4, 0xffff

; load lower 16 bits of 32-bit immediate
; using a preprocessor directive
LLI r5, %lo(0xffffffff)

; load a label
LLI r6, %lo(THIS_IS_A_LABEL)
~~~

### LUI - Load Upper Immediate
\iformat{0x06}

#### Desciption:
The `LUI` instruction loads the given 16-bit immediate into the upper 16 bits of the destination register `rd`, leaving the bottom 16 bits the same.
Combined with the `LLI` instruction, you can load an entire 32-bit immediate into a register.
Note that since this instruction does not touch the bottom 16 bits the register may not have the value you think it does if you do not know what is in the bottom bits.

#### Action:
~~~
rd[16..31] := imm16
~~~

#### Assembler Syntax:
~~~
LUI {rd}, {imm16}
~~~

#### Example:
~~~
; load a decimal immediate
LUI r0, 512

; load a hexadecimal immediate
LUI r1, 0x42

; load a binary immediate
LUI r2, 0b11001100

; load a character
LUI r3, 'X'

; load a 16-bit immediate
LUI r4, 0xffff

; load upper 16 bits of 32-bit immediate
; using a preprocessor directive
LUI r5, %hi(0xffffffff)

; load a label
LUI r6, %hi(THIS_IS_A_LABEL)
~~~

### MOV - Move
\dformat{0x07}

#### Desciption:
The `MOV` instruction simply copies the value of the source register into the destination register, keeping the value of the source register the same.

#### Action:
~~~
rd := rs
~~~

#### Assembler Syntax:
~~~
MOV {rd}, {rs}
~~~

#### Example:
~~~
; move the value of r7 into rv
MOV rv, r7
~~~
