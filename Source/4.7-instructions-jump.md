## Jump Instructioons

### JMP - Jump
\tformat{0x1F}

#### Desciption:
The `JMP` instructions simply _jumps_ to the address denoted by the value of the destination register and is used to modify control flow.

#### Action:
~~~
pc := rs
~~~

#### Assembler Syntax:
~~~
JMP {rd}
~~~

#### Example:
~~~
; jump to arbitrary address
LDI r0, THIS_IS_AND_ADDRESS
JMP r0
~~~

### JEZ - Jump If Zero
\tformat{0x20}

#### Desciption:
The `JEZ` instruction jumps to the address stored in the destination register only if the value of the source register is zero.

#### Action:
~~~
if rs = 0, pc := rd
~~~

#### Assembler Syntax:
~~~
JEZ {rd}, {rs}
~~~

#### Example:
~~~
; jump to arbitrary address
LDI r0, THIS_IS_AND_ADDRESS
LDI r1, 0
JEZ r0, r1
~~~

### JEQ - Jump If Equal
\tformat{0x22}

#### Desciption:
The `JEQ` instruction jumps to the address stored in the destination register only if the two source registers are equal.

#### Action:
~~~

#### Assembler Syntax:
~~~
JEQ {rd}, {rs0}, {rs1}
~~~

#### Example:
~~~
; jump to arbitrary address
LDI r0, THIS_IS_AND_ADDRESS

JEZ r0, r1, r2
~~~

### JGT - Jump If Greater Than
\tformat{0x24}

#### Desciption:
The `JGT` instruction jumps to the address stored in the destination register only if `rs~0~` is greater than `rs~1~`, interpreting the two values as signed.

#### Action:
~~~
if rs0 > rs1, pc := rd
~~~

#### Assembler Syntax:
~~~
JGT {rd}, {rs0}, {rs1}
~~~

#### Example:
~~~
; jump to arbitrary address
LDI r0, THIS_IS_AND_ADDRESS
LDI r1, 1
LDI r2, 0
JGT r0, r1, r2
~~~
